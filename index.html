1. Discuss the scope of `var`, `let`, and `const`

Ans: var is a function scope. means if wwe declare var inside a function, we can access that both inside,outside from function. It declares globally and can be accessed from anywhere. But let and const is blocked scope. means only can access that value inside function or block {}. And difference between let and const is, let can be redeclaredd, but const can't be.
